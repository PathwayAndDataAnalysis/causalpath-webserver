CausalPath
       CausalPath: Causal Interactions from Proteomic Profiles
Previous Next
CausalPath evaluates proteomic and other molecular measurements against our prior knowledge of cellular signaling and identifies potential causality among detected changes.  The method uses prior knowledge in high precision, distinguishes between activating and inhibiting phosphorylations, performs a site-specific evaluation of causality.  The output of the method is a network model of signaling that explains the observations.  Please refer to this article for a description of the method. 

A Java implementation of CausalPath is freely available.  We also provide this web service for easy access to the method and/or visualizing analysis results.  Please note the web server does not support network significance calculations in the analysis due to limited resources.  If that is desired, then the method needs to be run locally from its JAR file (see above link). Analysis results, however, can still be visualized on this web server using option C below. 

For a CausalPath analysis, please prepare your input files according to the guidelines.  If you already have results, please visualize them by uploading the related output folder(s) using option C below.  A sample dataset and analysis parameters can be found in this archive.  This can be directly uploaded for analysis through option B below.

The source code of this web server is freely available in a separate project.
